## ⚙️ Como rodar o projeto

```bash
# Clonar o repositório
git clone https://github.com/andressatomiozzo/react.git

# Entrar na pasta
cd 003-to-do-list

# Instalar dependências
npm install

# Rodar o projeto
npm run dev
```
